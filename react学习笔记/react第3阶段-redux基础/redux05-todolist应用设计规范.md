# 二、代码组织方式

按功能组织，一个功能包含若干角色。角色包括reducer、action等。每个功能模块对应一个目录。每个目录

包含不同的角色：

- actionTypes.js,定义action类型
- action.js，定义action构造函数
- reducer.js,state和action的对应关系
- views目录：react组件
- index.js,所有角色的导入
