> 解析解构赋值，是一种快复杂数据类型速赋值的方式。

#### 1、数组解构赋值

如果解构的是数据，那么数组中的下标对应的数据会与等号左边的值一一对应

```js
 var [a,c,b] = [1,2,3];
```

#### 2、对象解构赋值

  如果解构的是对象，那么在解构时，会以属性名为标准进行赋值

```js
var {name,age} = {age: 19, name: "张三"}
```
在对对象进行解构时，可以给属性设置别名(一旦设置别名，就不能直接通过属性名来取值了)

```js
var {name:nickName} = {age: 19, name: "张三"};
```
