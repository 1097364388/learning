> 数组方法


### 一、添加删除

#### 1、unshift()和push()

【改变原数组】

**unshift()**：从数组的头部插入一个或者多个元素，并返回新数组的长度。

**push()**：向数组的尾部添加一个或多个元素，并返回新数组的长度。

```javascript
 var arr = [1, 2, 3];
 console.log(arr); //[1,2,3]
 var b = arr.unshift(0,1);// [0, 1, 1, 2, 3]
```

#### 2、shift()和pop()

【改变原数组】

**shift()**:从数组头部移除一个元素。

**pop()** : 从数组尾部移除一个元素。

```javascript
var arr = [1, 2, 3];
console.log(arr); //[1,2,3]
var b = arr.shift();
console.log(arr);// [2,3]
```

### 二、位置方法

#### 1、indexOf()

1) 返回指定元素在数组中出现的位置，如果没有出现则返回-1

```javascript
var arr = [1, 2, 3];
var b= arr.indexOf(2);
var c= arr.indexOf(4);
console.log(b);//1
console.log(c);//-1
```

2) indexOf()还可以接受第二个参数表示搜索的开始位置
```javascript
var arr = [1, 2, 3];
var b= arr.indexOf(1,1);
console.log(b);// -1
```

### 三、转换为字符串

#### 1、toString()
将数组平铺为字符串，以逗号分隔。
```javascript
var arr = [1, 2, 3,[4,5,6]];
console.log(arr.toString());//"1,2,3,4,5,6"
```

#### 2、join()
一个数组（或一个类数组对象）的所有元素按指定字符连接成一个字符串并返回这个字符串。默认以逗号连接与toString()方法相同。

```javascript
var a = ['Wind', 'Rain', 'Fire'];
var myVar1 = a.join();      // myVar1的值变为"Wind,Rain,Fire"
var myVar2 = a.join(', ');  // myVar2的值变为"Wind, Rain, Fire"
var myVar3 = a.join(' + '); // myVar3的值变为"Wind + Rain + Fire"
var myVar4 = a.join('');    // myVar4的值变为"WindRainFire"
```

###  四、数组操作
#### 1、concat 
功能：合并两个或多个数组。

【不改变原数组】
```javascript
var arr1 = [1, 2, 3];
var arr2 = [4,5,6];
var b=arr1.concat(arr2);
console.log(b);//[1, 2, 3, 4, 5, 6]
```

#### 2、reverse()
功能：数组中元素的位置颠倒。
【改变原数组】
```javascript
var arr = [1, 2, 3,4,5,6]
console.log(arr.reverse());//[6, 5, 4, 3, 2, 1]
```

#### 3、slice()
返回一个从开始到结束（不包括结束）选择的数组的一部分浅拷贝到一个新数组对象。且原始数组不会被修改。
【不改变原数组】

第一个参数：截取数组的开始位置
第二个参数：截取数组的终止位置（不包含此位置的数据）
如果只写一个参数表示截取到最后
```javascript
var arr = [1, 2, 3,4,5,6]
console.log( arr.slice(0));// [1, 2, 3, 4, 5, 6]
console.log( arr.slice(1,2));// [2]
console.log( arr.slice(1));// [2, 3, 4, 5, 6]
console.log(arr)//[1, 2, 3, 4, 5, 6]原数组不变
```

#### 4、splice()

功能：方法通过删除现有元素和/或添加新元素来更改一个数组的内容。

【改变原数组】

第一个参数时删除的起始位置，第二个参数时删除的个数，后面的参数为新增的元素

```javascript
var arr = [1, 2, 3,4,5,6]
var b=arr.splice(4,2,8,8)
console.log(b);//[5, 6]从下标为4的位置删除俩个成员
console.log(arr);//[1, 2, 3, 4, 8, 8]从下标为4的位置删除俩个成员，有插入俩个成员
console.log(arr.splice(-4,2));//[3, 4]起始位置如果未负数表示从倒数位置开始删，此位置也等于数组的长度减去这个负数
console.log(arr.splice(1,0,8));//[]如果只插入元素，第二个参数可以设为0
console.log(arr);//[1, 8, 2, 8, 8]如果只插入元素，第二个参数可以设为0
console.log(arr.splice(2));//[2, 8, 8]如果只有一个参数，相当于将原数组在制定的位置拆分成俩个数组
console.log(arr);//[1, 8]//该方法将改变原数组
```

### 5排序方法（reverse也是排序方法）

#### 11、sort():对数组进行排序，默认是按照字典顺序排序。排序后，原数组将被改变。

```javascript
//1字符串排序
var arr=['el','g','q','dd'，'eg','dt']
console.log(arr.sort());//["dd", "dt", "eg", "el", "g", "q"]
//2对数字进行排序
var arr=[11,101]
console.log(arr.sort());//[101, 11]可以看出它并没有按照数字大小进行排序，而是按照字符串的方式（先比较第一位，如果第一位相等依次比较后面的）

//3对数字进行大小排列-------------------------------------------------
//如果想让sort方法按照自定义方式排序，可以传入一个函数作为参数，表示按照自定义方法进行排序。该函数本身又接受两个参数，表示进行比较的两个元素。如果返回值大于0，表示第一个元素排在第二个元素后面；其他情况下，都是第一个元素排在第二个元素前面。
function arrNum(a,b){
  return a-b//从小到大
}
var arr=[10,31,24,62,101,96]
console.log(arr);//[10, 31, 24, 62, 101, 96]原数组将被改变
console.log(arr.sort(arrNum));//[10, 24, 31, 62, 96, 101]

//4同时按照两种方式排序。需求是这样的：“active”为1的排在前面，“active”为0的排在后面，然后分别按照“name”的字母顺序进行排序：
var arr= [{
    "name"  : "Alfredo",
    "active": 0
}, {
    "name"  : "Alexander",
    "active": 0
}, {
    "name"  : "Ronald",
    "active": 1
}, {
    "name"  : "salvador",
    "active": 0
}, {
    "name"  : "Saryn",
    "active": 1
}];

arr.sort(function (a, b) {
    return b.name < a.name;
}).sort(function (a, b) {
    return b.active - a.active
});
console.log(arr);
//0: {name: "Ronald", active: 1}
//1: {name: "Saryn", active: 1}
//2: {name: "Alexander", active: 0}
//3: {name: "Alfredo", active: 0}
//4: {name: "salvador", active: 0}
```

#### 12map()对数组所有成员依次调用一个函数，根据函数结果返回一个新数组,原数组没变化

```javascript
var arr = [1, 2, 3,4,5,6]
var b=arr.map(function(n){
  return n+1
})
console.log(b);// [2, 3, 4, 5, 6, 7]
```

#### 13filter()参数是一个函数 所有数组成员依次执行该函数，返回结果为true的成员组成一个新数组。该方法不会改变原数组

```javfascript
var arr = [1, 2, 3,4,5,6]
var b=arr.filter(function(n){
  return n>1
})
console.log(b);// [2, 3, 4, 5, 6]
```

#### 14、for each()

array.forEach(function(currentValue, index, arr), thisValue) 
方法：遍历数组，传入一个回调函数，里面有三个参数，第一个是当前元素值，第二个是索引，第三个是数组对象。thisValue可选。传递给函数的值一般用“this”值，如果这个值为空，“undefined”会传递给this值

```javascript
var arr = [1, 2, 3,4,5,6]
arr.forEach(function(item,index,arr){
  console.log(item,index);
})	
```

#### 15、every

传入一个回调函数，返回值是Boolean

要求每一个值都符合条件

```javascript
const arr = [1, 2, 3,4,5,6]
const  a = arr.every(function(e){
  return e>3
})
console.log(a);//false
```

#### 16、some()

传入一个回调函数，返回值是Boolean

只要有一个值都符合条件

```javascript
const arr = [1, 2, 3,4,5,6]
const  a = arr.some(function(e){
  return e>3
})
console.log(a);//true
```

### 操作方法

#### 1、contact

连接字符串返回一个新字符串，对原有字符串没有任何改变，一般不用因为一个+就能解决

```javascript
var str='qwe';
var str1='abc';
var str2=str.concat(str1);
```

#### 2slice（n,m)

从索引n开始截取到索引m,不包括m,将截取的字符串返回，对原字符串没有任何改变

与数组一样

```javascript
var str = "hello world!"
var str1= str.slice(2,3)
console.log(str);//hello world!
console.log(str1);//l
```

#### 3**substring**

与slice一样

```javascript
var str = "hello world!"
var str1= str.substring(2,3)
console.log(str);//hello world!
console.log(str1);//l
```

#### 4**substr**类似splice,截取片段

从索引n开始截取m个字符,将截取的字符返回,对原符串没有任何改变

```javascript
var str = "hello world!"
var str1= str.substr(2,3)
console.log(str);//hello world!
console.log(str1);//llo
```

### 位置

#### indexof()

判断一个字符第一次出现在某个字符串的索引,如果包含返回它的索引,如果不包含返回-1

```javascript
var str = "hello world!"
console.log(str.indexOf('l'));//2
console.log(str.indexOf('f'));//-1
```

#### .lastIndexOf 判断一个字符最后一次出现在某个字符串的索引，如果包含返回它的索引，如果不包含返回-1

```javascript
var str = "hello world!"
console.log(str.lastIndexOf('l'));//9
console.log(str.lastIndexOf('f'));//-1
```

### 检索

#### **match()**

可在字符串中检索指定的值,或找到一个或多个正则表达式匹配.把找到的字符放到组里返回

```javascript
var str = "hello world!"
console.log(str.match('l'));//["l", index: 2, input: "hello world!", groups: undefined]
```



#### search() 方法用于检索字符串中指定的子字符串，或检索与正则表达式相匹配的子字符串。

```java
var str = "hello world!"
console.log(str.search('l'));//2
```
